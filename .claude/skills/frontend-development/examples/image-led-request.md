- 首屏主任务：快速定位供应商并进入配置 / 编辑

- 页面目标：快速查看、配置和管理模型供应商
- 主要对象：模型供应商
- 核心动作：搜索供应商、查看状态、配置 / 编辑 / 测试连接
- 页面结构：左侧设置导航，顶部标题与搜索，中部已安装列表，下部推荐安装区
- 关键模块：提示条、供应商卡片、状态标签、主操作区、推荐区

- 关键状态：loading、empty、error、success、running、draft
- 视觉约束：浅底工作台、暖灰边框、翡翠绿高信号、非营销式排版
